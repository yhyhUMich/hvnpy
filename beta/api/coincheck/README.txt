# vn.coincheck

### 简介

coincheck 的比特币交易接口，基于Rest API开发，实现了官方提供API的全部功能。




### 特点

1. 面向对象的API设计，接近CTP API的结构，对于国内用户而言更容易上手

2. 参考CTP API的设计，主动函数调用的结果通过异步（回调函数）的方式推送到程序中，适用于开发稳定可靠的实盘交易程序
